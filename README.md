## 🚀 Quick Start

### 1. Install Dependencies

```bash
npm install react react-dom
# or
yarn add react react-dom
```

### 2. Use the Component

```jsx
import Hero from './components/Hero/Hero'

function App() {
  return (
    <div>
      <Hero />
      {/* Your other sections */}
    </div>
  )
}

export default App
```

### 3. Import CSS

Make sure to import the CSS files in your main app or component:

```jsx
import './components/Hero/hero.css'
import './components/Ui/container.css'
import './components/Ui/button.css'
```

## 🎯 Key Optimizations

### Performance ⚡
- **React.memo()** - Prevents unnecessary re-renders
- **requestAnimationFrame** - Smooth 60fps animations
- **Passive event listeners** - Better scroll performance
- **GPU acceleration** - All animations use transform/opacity
- **CSS containment** - Isolated paint/layout calculations

### Design 🎨
- **Gradient orbs** - Animated background elements
- **Glassmorphism** - Modern frosted glass effects

## 🎨 Customization

Easy to customize to match your brand:

### Change Colors
```css
.gradient-text {
  background: linear-gradient(135deg, #YourColor1 0%, #YourColor2 100%);
}
```

### Update Copy
```jsx
<h1 className="hero-title">
  <span className="title-line">Your First Line</span>
  <span className="title-line gradient-text">Your Second Line</span>
</h1>
```

### Modify Stats
```jsx
<div className="stat-item">
  <span className="stat-number">Your Number</span>
  <span className="stat-label">Your Label</span>
</div>
```
